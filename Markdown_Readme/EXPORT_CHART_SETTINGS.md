# 导出图设置（Excel 风格能力）

> 文档定位：导出图设置能力说明与导出样式规范。
> 返回总览：[ROADMAP（总览导航）](./ROADMAP.md)

## 目标

将图表导出配置集中到统一入口，支持在导出前快速调整图形格式，覆盖常见 Excel 图表格式化需求。

## 使用入口

任意支持 PNG 导出的图表下方，展开“导出图设置”。

## 当前支持项

- 网格线控制
  - 显示 X 网格线
  - 显示 Y 网格线
  - 网格线颜色
- 坐标轴控制
  - 显示坐标轴线
  - 坐标轴线颜色
- 图例控制
  - 显示图例
  - 图例位置（右侧/顶部/底部/左侧）
- 系列配色
  - 保留原图配色
  - Plotly / Safe / Set2 / Pastel / Dark24
  - 单系列手工改色（逐图例项）
- 标题与轴标题
  - 导出标题（可选覆盖）
  - X 轴标题（可选覆盖）
  - Y 轴标题（可选覆盖）
- 轴刻度格式
  - X 轴刻度格式（原始/千分位/百分比/科学计数法）
  - Y 轴刻度格式（原始/千分位/百分比/科学计数法）
  - 小数位控制（0~4）
- 画布与字体
  - 导出宽度/高度（px）
  - 字体大小
  - 整体背景色
  - 绘图区背景色
- 数据标签（全局联动）
  - 显示模式：关闭 / 仅数值 / 仅系列名 / 仅Model / 系列名+数值
  - 标签位置：自动 / 内侧 / 外侧 / 顶部 / 中间
  - 支持图型：条形图、散点/气泡图、热力图、饼图等

## 设计原则

- 默认保守：保持原图配色，避免影响业务图既有颜色语义。
- 导出一致：导出样式直接作用于导出图对象，预览与导出保持一致。
- 类型安全：连续色阶图（如热力图）不强制应用离散色板。
