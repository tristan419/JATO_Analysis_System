# 图表导出 PNG 评估（v1）

> 文档定位：PNG 导出能力选型、风险评估与实现结论。
> 返回总览：[ROADMAP（总览导航）](./ROADMAP.md)

## 目标

在 Dashboard 中支持高质量图表导出（PNG），用于报告与演示。

## 备选方案

### 方案 A：Plotly + Kaleido（推荐）

- 方式：`fig.to_image(format="png")`
- 依赖：`kaleido`
- 优点：接入简单、图形质量高
- 风险：部署环境需额外安装依赖

### 方案 B：浏览器截图

- 通过前端截图导出
- 优点：不加后端依赖
- 风险：不同设备分辨率差异较大

## 推荐结论

优先采用方案 A，后续在 `views.py` 增加“导出当前图 PNG”按钮。

## 导出图设置升级（v2）

- 导出入口文案统一为“导出图设置”。
- “生成PNG”按钮移入“导出图设置”折叠面板。
- 支持导出级样式配置（参考 Excel 常见图表格式能力）：
	- 网格线：X/Y 网格线开关、网格线颜色
	- 坐标轴：坐标轴线开关、轴线颜色
	- 图例：显示开关、位置（右侧/顶部/底部/左侧）
	- 配色：原图配色、Plotly、Safe、Set2、Pastel、Dark24
	- 单系列手工改色：逐图例项指定导出颜色
	- 轴刻度格式：X/Y 轴独立选择（原始/千分位/百分比/科学计数法）+ 小数位
	- 标题与轴标题：支持导出时覆盖标题文本
	- 画布：导出宽高、背景色、绘图区背景色
	- 数值标签：条形图导出时可显示数值

## 兼容性策略

- 默认选项为“保留原图配色”，避免破坏业务图既有配色语义。
- 导出样式在图像导出链路统一应用，确保“所配即所导”。
- 热力图等连续色阶图不强制套用离散色板，避免失真。
